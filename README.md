# Data-cleaning
HW1 for Data Mining

## 数据集描述：  
本数据关于 248 辆出租车在某一天的运营情况(大约每 60 秒一个数据点)

## 数据分析目标：
- 将原始数据汇总成为订单级别的数据

- 清洗异常值
   - 清洗订单层面的异常值
   - 清洗司机层面的异常值
   
- 汇报并比较清洗前后的如下指标：
  - 司机总数
  - 订单总数
  - 司机平均每天的订单数
  - 司机平均每天的行驶里程
  - 司机平均每天的收入金额
  - 司机平均每天的载客营运时长
  - 司机平均每天的工作时长(载客+空驶，不考虑停运时间)
  - 司机平均每天的空驶时间比例(空驶/[载客+空驶])
 
本次数据清洗采用 **R语言**

详细内容阐释见 **hw1.pdf**
